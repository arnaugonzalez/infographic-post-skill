In `scripts/generate_posts.py`, the `__main__` block at the end of the file reads:

```python
if __name__ == "__main__":
    _build_parser().parse_args()
    main()
```

The `_build_parser().parse_args()` on the first line of the block is dead code — its return value is discarded. `main()` (line 229) calls `_build_parser()` and `parser.parse_args()` itself and uses the result. Argv is parsed twice; the first parse is wasted.

Fix: delete the `_build_parser().parse_args()` line so the block becomes:

```python
if __name__ == "__main__":
    main()
```

No other changes to the file. Do not modify main(), _build_parser(), or any other function.
